 # ReactJS Internship Project

A foundational project focused on building modular, component-based user interfaces and exploring the React lifecycle. This repository serves as a baseline for my transition into professional frontend architecture and state management.

## 🎯 Project Goals
* Implement a clean **Component-Based Architecture** to ensure UI reusability.
* Manage application state and props flow effectively across multiple views.
* Explore the integration of CSS and JavaScript modules for responsive design.
* Deploy a functional frontend via GitHub Pages.

## 🛠️ Technical Stack
* **Core:** React.js
* **Styling:** CSS3 (Modular styles)
* **Language:** JavaScript (ES6+)
* **Deployment:** GitHub Pages / Jekyll integration

## 🏗️ Key Features
* **Modular UI:** Reusable components for navigation and page layouts.
* **Multi-page Navigation:** Implementation of basic routing and page structure (Home, About Us).
* **Responsive Layout:** Integrated CSS to handle cross-device consistency.

## 🚀 Deployment & Setup

This project is deployed using GitHub Pages. To run it locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/sai-surya-velaga/reactjs-internship.git](https://github.com/sai-surya-velaga/reactjs-internship.git)
    ```
2.  **Open the project:**
    Open `index.html` in your browser or run a local development server.
